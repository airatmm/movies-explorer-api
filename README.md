# Movies explorer (API)
Дипломный проект
### Сервис для поиска фильмов с возможностью сохранить их в личном кабинете. 

## Стек
* Node.js
* Express
* MongoDB

## Запуск проекта локально

1. Клонировать проект
2. Установить зависимости
```
npm i
```
3. Запустить проект
```
npm start
```
Проект запустится по адресу http://localhost:3000/
